================================================================================
FOLDER DESCRIPTIONS
================================================================================

simple_downloaders/
  Fast, one-shot downloaders for quick data grabs.
  - No monitoring, just download and save
  - Headless mode (CSV) or direct HTTP (RSS)
  - Perfect for cron jobs or scheduled tasks

batch_scripts/
  Windows batch files for double-click simplicity.
  - Just double-click to download
  - Calls Python scripts in simple_downloaders/
  - Great for non-technical users

monitors/
  Advanced tracking scripts for long-term monitoring.
  - Run for 60 minutes (customizable)
  - Detect and log all updates
  - Generate statistics and summaries
  - Perfect for research and analysis

utilities/
  Helper scripts and parsers.
  - RSS feed parser
  - Can be imported into your own scripts
  - Reusable components

downloads/
  Output folder for all downloaded data.
  - Auto-created if doesn't exist
  - All scripts save here
  - Includes CSV, XML, and log files

Data Richness        | High                 | Medium + News
Download Speed       | ~10 sec              | ~1 sec
Browser Required     | Yes (headless)       | No
File Size            | ~140 KB              | ~20 KB
Best For             | Deep analysis        | Quick checks
Script Location      | simple_downloaders/  | simple_downloaders/

Recommendation:
  â†’ Use CSV for comprehensive monitoring
  â†’ Use RSS for quick news snapshots

FILE NAMING CONVENTIONS
================================================================================

RSS:
  trending_rss_YYYYMMDD_HHMMSS.xml
  Example: trending_rss_20251102_041043.xml

Logs:
  google_trends_log_YYYYMMDD_HHMMSS.log
  Example: google_trends_log_20251102_014416.log

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "ChromeDriver not found"
  â†’ Download from: https://chromedriver.chromium.org/downloads
  â†’ Match your Chrome version (check chrome://version)
  â†’ Add to PATH or place in project folder

Issue: "Could not click Download CSV option"
  â†’ Google may have changed their UI
  â†’ Script tries 4 different click methods
  â†’ Check downloads/debug_screenshot.png if generated

Issue: "No new file detected"
  â†’ Verify downloads/ folder exists
  â†’ Check Chrome default download location
  â†’ Ensure write permissions

Issue: Scripts can't find files after reorganization
  â†’ Run batch files from project root directory
  â†’ Or use full paths: py C:\...\simple_downloaders\download_trends_simple.py

Issue: RSS works but CSV doesn't
  â†’ Normal! RSS uses simple HTTP (more reliable)
  â†’ CSV requires browser automation (more fragile)
  â†’ Use RSS as backup if CSV fails
